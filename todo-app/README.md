# Todo App

このプロジェクトは、基本的なTodoアプリを作成するためのものです。以下に、アプリの構成と使用方法を説明します。

## 構成

- `index.html`: Todoアプリの基本的なHTML構造を持つファイルです。アプリのタイトル、入力フィールド、Todoリストを表示するための要素が含まれています。
- `src/js/app.js`: アプリの主要なJavaScriptロジックを含むファイルです。Todoアイテムの追加、削除、完了状態の管理などの機能を実装しています。
- `src/css/styles.css`: アプリのスタイルを定義するファイルです。レイアウト、色、フォントなどのスタイルが含まれ、アプリの見た目を整えます。
- `package.json`: npmの設定ファイルで、プロジェクトの依存関係やスクリプトをリストしています。

## セットアップ手順

1. リポジトリをクローンまたはダウンロードします。
2. プロジェクトディレクトリに移動します。
3. 必要な依存関係をインストールするために、以下のコマンドを実行します。

   ```
   npm install
   ```

4. アプリを起動するために、以下のコマンドを実行します。

   ```
   npm start
   ```

## 使用方法

1. アプリを起動すると、Todoリストが表示されます。
2. 入力フィールドに新しいTodoアイテムを入力し、追加ボタンをクリックします。
3. Todoアイテムがリストに追加されます。
4. 各アイテムの横にある削除ボタンをクリックすると、そのアイテムを削除できます。
5. 完了したアイテムは、チェックボックスをクリックすることで完了状態にできます。
